request get : 
			GET방식은 서버로부터 정보를 조회하기 위해 설계된 메소드
			데이터를 쿼리스트링을 통해서 전송하다.
				쿼리스트링 : 이름과 값의 쌍으로 이루어져있다.
						 여러 값을 보내야 할 경우에는 &로 이어준다.
						 ex)?name=hmd&gender=male
				URI에 데이터 노출되어 보안에 취약하다.
				전송하는 데이터의 길이에 제한이 있다.

request : post

			POST : 전송 할 데이터를 HTTP 메세지의 BODY에 담아서 전송
			전송하는 길이에 제한이 없다.
			POST 방식을 사용할 때는 요청 헤더에 Content-Type을 표시해야 한다.

			form으로 전송할 경우 기본값으로 application/x-www-form-urlencoded
			방식으로 지정되어 있다.
			form의 enctype 속성을 통해서 변경이 가능하다.
					multipart/form-data 로 변경 가능.


			HTTP메세지
			시작줄	: 1)HTTP METHOD 2) URI 주소 3) http버전
			Header  : 
			body    : 데이터

request:post enctype="multipart/form-data"
