1. block/inline 요소

block: 좌우영역을 부모요소의 전체 크기만큼 잡는다. 상하영역은 컨텐츠의 크기만큼 잡는다.
inline : 상하좌우영역 모두 컨텐츠의 크기만큼 잡는다. inline : 상하좌우영역 모두 컨텐츠의 크기만큼 잡는다.

2. block/inline 요소의 height, width

block : width와 height를 사용하여 크기를 지정할 수 있다.
inline : width와 height를 사용하여 크기를 지정할 수 없다.

3. block/inline 요소의 margin과 padding

block : margin과 padding이 상하좌우 모두 적용된다.
inline : margin과 padding이 좌우 모두 적용된다.
block : margin과 padding이 상하좌우 모두 적용된다.
