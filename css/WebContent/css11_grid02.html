float속성 활용하기

block : 한 줄에 여러개의 block 요소를 배치하고 싶다면? float:left
float:left속성이 적용된 요소는 좌측으로 정렬된다. float속성은 보통흐름(normal flow)에서 해당 요소를 제외시킬수 있다.
float: left
float: left
float: left
float: right
	overflow:hidden 은 본래 영역을 벗어난 컨텐츠를 숨기는 역할을 하는 속성이지만 
	float 속성을 clear 해주기도 한다.

display:inline-block

block요소는 display 속성 값이 block 이다. display : inline-block 이라는 속성은 해당 요소를 inline요소 처럼 다루면서 width와 height의 크기를 지정할 수 있도록 한다.
display:inline-block
