block요소 위치 잡기 / top, bottom, left, right

position : 요소들의 위치를 잡아주는 css속성
static : 기본값, 다른 태그들과의 관계에 의해서 자동으로 배치 위치를 임의로 변경할 수 없다.
relative : 다른 태그들과의 관계에 의해서 자동으로 배치 위치를 임의로 변경할 수 있다. absolute : 부모요소의 좌측 상단부터 위치를 정렬한다.
fixed : 스크롤과 상관없이 보이는 화면의 위치를 기준으로 위치를 지정
top, bottom, left, right
지정한 반향에서 안쪽으로 지정한 크기만큼 요소의 위치를 이동시킨다.
%로 크기를 지정하면 부모요소의 크기의 %만큼 이동한다. 부모요소에 height 속성이 없으면 정상작동하지 않는다.

float:left; position:absolute; top: 40%; position: absolute : 부모요소의 좌측 상단에 요소를 배치한다. 부모요소의 좌측상단으로 부터 부모요소의 세로길이의 40%에 해당하는 위치에 요소를 배치 부모요소의 position이 static이면 안된다.
