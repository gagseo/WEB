URL 인코딩, URL 디코딩 함수

		URL 인코딩
		인터넷에서 URL은 아스키 문자열로 표현가능하게끔 구성되어야 한다.
		아스키코드로 표현이 불가능한 문자(한글, 일본어, 중국어....)
		를 브라우저에서 표현가능한 형태로 인코딩해주는 함수를
		자바스크립트가 제공해준다.

		escape : 안씀
		encodeURI : (:, ;, /, =, ?, &) 제외하고 인코딩
		encodeURIComponent : 알파벳과 숫자를 제외한 모든 문자


eval() 함수

		문자열을 자바스크립트 코드로 변환해 실행하는 함수
