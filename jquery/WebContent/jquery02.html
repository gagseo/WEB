속성선택자

기본선택자 뒤에 붙여 사용한다.

요소[속성]
요소[속성=값] : 속성의 값으로 객체를 선택
요소[속성~=값] : 속성 안의 값이 특정 값과 같은 객체를 선택
요소[속성^=값] : 속성 안의 값이 특정 값으로 시작하는 객체를 선택
요소[속성$=값] : 속성안의 값이 특정 값으로 끝나는 객체 선택
요소[속성*=값] : 속성안의 값이 특정 값을 포함하는 객체 선택
